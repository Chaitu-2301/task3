# Task 5 - Weather App

## 📌 Overview
A **Weather App** built with HTML, CSS, and JavaScript.  
It can show static weather details (offline) or fetch live weather data using the **OpenWeather API**.

## ✨ Features
- Search by city name  
- Display temperature, condition, humidity, wind  
- Error handling for invalid input  
- Responsive design  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- (Optional) OpenWeather API  

## 🚀 How to Run
1. Download or clone this repository.  
2. Open `index.html` in your browser.  
3. (For live weather) Get a free API key from [OpenWeather](https://openweathermap.org/) and replace `YOUR_API_KEY` in the code.
